# Real-estate-occupancy-analysis
Data analysis and dashboarding project analyzing HVAC performance and occupancy trends for smart buildings using Python and Tableau.
# Smart Building Analytics - KODE Labs Assignment

## Overview

This project analyzes HVAC temperature performance and building occupancy data to generate operational and executive insights for smart building management.

## Objectives

### Problem 1
Analyze hourly temperature performance during Occupied vs Unoccupied periods.

### Problem 2
Calculate the time required for HVAC systems to reach target temperature setpoints after occupancy begins.

### Problem 3
Design dashboards for:
- Building Operators
- Executives

## Tools Used

- Python (Pandas, NumPy)
- Google Colab
- Tableau Public
- SQL concepts

## Key Insights

- Occupied spaces maintained temperatures closer to target setpoints.
- Average portfolio utilization was approximately 9.21%.
- Dashboarding identified occupancy trends, top-performing buildings, and utilization opportunities.
